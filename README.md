📌 Project Overview
EMS-MiniProject2 is a full-stack Employee Management System built using:

Backend: ASP.NET Core Web API (.NET 8)
Frontend: HTML, CSS, Vanilla JavaScript
Testing: NUnit + Moq
This project is an upgraded version of Mini Project 1 with full backend integration, authentication, employee CRUD operations, dashboard analytics, and unit testing.

🚀 Features
🔐 Authentication
User Login
JWT Token Authentication
Secure API access
👨‍💼 Employee Management
Add Employee
Update Employee
Delete Employee
View Employee by ID
View All Employees
Pagination support
📊 Dashboard
Total Employees count
Active Employees count
Department-wise analytics
Live UI updates
🧪 Testing
NUnit test cases
Moq mocking for services
AuthService unit tests
EmployeeService unit tests
